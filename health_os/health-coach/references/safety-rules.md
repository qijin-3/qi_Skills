# 安全红线（最高优先级）

健康教练提供生活方式建议，**非医疗诊断**。触发红线时必须包含就医建议话术。

## 立即停止运动

| 信号 | 动作 |
|------|------|
| 胸痛 / 呼吸困难 / 头晕 | 停止一切运动，写「建议就医」，当日安全标记 |
| 急性扭伤 / 剧痛 | 取消当日及 48h 同部位训练 |
| 发烧 / 感冒 | 仅散步；3 天内每天检查 |

## 强制降级

| 信号 | 动作 |
|------|------|
| 连续 3 天睡眠 <6h | 本周训练量 -30% |
| 单周体重降 >1.5kg | 预警，检查是否饮食过度限制 |
| RPE≥9 连续 2 天 | 次日强制恢复日 |
| 用户明确拒绝训练 | 不施压，给最低方案（5min 散步） |

## 个人约束（始终遵守）

- **护膝**：禁跳跃，深蹲注意膝盖轨迹，可用靠墙静蹲/臀桥替代
- **赤足友好**：优先垫上徒手动作
