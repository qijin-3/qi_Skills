# Mode B：年度复盘

> 适用于：用户说"年度复盘"、"做年度总结"，或由调度系统在年初触发。
> 前提：`north-star.md` 已存在。若不存在 → 切换到 Mode A。

**执行前必读**：Read `{PERSONAL_OS_ROOT}/content/about-me.md` — 全年数据分析时，结合「需要警惕的时间陷阱」章节，判断哪些陷阱今年反复出现、哪些已成功避免；同时为方向检验问题提供背景。

---

## Step 1：拉取全年客观数据

**1a. 全年完成任务汇总**

```bash
lark-cli task +get-my-tasks --as user \
  --complete=true \
  --format json \
  --page-all
```

过滤 `completed_at` 属于上一年度（`{YYYY-1}`）的任务，汇总：
- 全年完成总数
- 清单分布：Writing / Growing / Projects / Routine / Idea
- 月度分布（12 个月各完成几条）

**1b. 月度完成率趋势（来自日志表）**

读 `TABLE_LOGS` 全年记录，提取每月 `今日任务完成度`（`FIELD_LOG_COMPLETION_RATE`）均值。
- 高效月份 = 月均完成度 ≥ 0.7
- 低效月份 = 月均完成度 ≤ 0.4

**1c. 北极星三支柱各完成度**

读 `TABLE_MONTHLY`，过滤 `周期包含{YYYY-1}`，按关联的 `北极星支柱`（link 字段）分组，计算各支柱月度目标平均 `目标完成度`。

**1d. 日记能量信号（可选，若 diary.md 存在）**

扫描上年各月 `{PERSONAL_OS_ROOT}/content/{YYYY-1}/{MM}/diary.md`，提取频繁出现的情绪/状态词，概述高效月份和低效月份的规律（≤3 句）。

---

## Step 2：发全年复盘摘要 + 方向检验提问

```
📊 {YYYY-1} 年度复盘

全年完成 {N} 条任务
  Writing:{n} | Growing:{n} | Projects:{n} | Routine:{n} | Idea:{n}

📈 月度完成率趋势：
  高效月份：{月份列表}（均值 {X}%）
  低效月份：{月份列表}（均值 {X}%）

🌟 北极星支柱推进情况：
  {支柱1}：年均完成度 {X}%
  {支柱2}：年均完成度 {X}%
  {支柱3}：年均完成度 {X}%

{若有日记数据} ⚡ 能量信号：{高效/低效月份规律，2-3 句}

---

方向检验（请逐一回答）：

1. 去年花时间最多的是什么？它今年还值得继续吗？
2. 去年哪件事给你带来了最强的心流体验？
3. 如果只能保留一个方向，你会选什么？
```

**等待用户逐一回答**，不催促，不汇总替代。

---

## Step 3：展示当前北极星 + 检验

Read `{PERSONAL_OS_ROOT}/content/north-star.md`，展示三支柱和衡量标准，结合全年数据：

```
当前北极星三支柱：
  - {支柱1}（去年完成度 {X}%）→ 目标：{年底标准}，实际：{客观评估}
  - {支柱2}（去年完成度 {X}%）→ 目标：{年底标准}，实际：{客观评估}
  - {支柱3}（去年完成度 {X}%）→ 目标：{年底标准}，实际：{客观评估}

基于你的回答和数据：
  {AI 的判断：哪个支柱仍然有效？哪个可能需要调整？给出理由，不是结论}

问题：去年的北极星方向还对吗？有什么想法？
```

**等待用户回应**。

- 用户表示方向无需调整 → 直接跳到 Step 5
- 用户表示需要调整 → 切换到 Mode C（Read `mode-c-adjust.md`）进行调整流程，完成后回到 Step 5

---

## Step 4：若需要，为新年起草北极星更新

仅当用户确认需要调整时执行（否则跳过）。具体流程在 `mode-c-adjust.md` 中。

---

## Step 5：写年度复盘文档

写入 `{PERSONAL_OS_ROOT}/content/{YYYY-1}/annual-review.md`：

```markdown
# {YYYY-1} 年度复盘

## 客观数据回顾

- 全年完成 {N} 条任务（Writing:{n} / Projects:{n} / Growing:{n} / Routine:{n}）
- 月度完成率趋势：高效月 {月份} | 低效月 {月份}
- 北极星支柱完成度：
  - {支柱1}：{X}%（目标：{年底标准}）
  - {支柱2}：{X}%（目标：{年底标准}）
  - {支柱3}：{X}%（目标：{年底标准}）

## 方向检验

| 支柱 | 完成度 | 关键成就 | 主要障碍 |
|------|--------|----------|----------|
| {支柱1} | {X}% | {用户提到的成就} | {障碍} |
| {支柱2} | {X}% | {成就} | {障碍} |
| {支柱3} | {X}% | {成就} | {障碍} |

## 能量信号（来自日记）

{高效月份规律 / 低效月份规律}

## About Me 更新提议

{若识别到≥3次/季度的稳定新模式，在此列出；否则写"无"}

## 北极星检验结论

{方向延续 / 局部调整，一句话总结}

## 下一年北极星

{若有更新：见 content/north-star.md（已于 {YYYY-01-01} 更新）}
{若无变化：沿用现有北极星，见 content/north-star.md}
```

---

## Step 6：输出复盘摘要

```
🌟 {YYYY-1} 年度复盘完成

{若北极星更新：北极星已更新，核心方向：{方向}}
{若无更新：北极星方向延续}

年度复盘文档：content/{YYYY-1}/annual-review.md
北极星：content/north-star.md
```
