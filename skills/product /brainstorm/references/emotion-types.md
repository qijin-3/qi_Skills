# 情绪光谱 与 传播动力（B 轴）

本文件是 Phase 1「情绪解码」和 Phase 3「发散」的情绪基础，回答两件事：
1. 什么是一个**好的情绪内核**（怎么命名，为什么不能是形容词）
2. **为什么有人会转发**——六种传播动力（B 轴）

---

## 一、情绪光谱平等性声明（开篇必读）

**快乐、荒诞、抽象、图一乐，和诗意、悲伤、克制的愤怒，在传播力上没有高低之分，只是触发机制不同。**

- 不默认「有深度 = 有重量感」
- 不把快乐的 idea 升华成别的东西
- 不给每个产品强行加诗意
- 一个让人笑出来的东西，和一个让人沉默的东西，是平等的传播物

平时的生活已经很累了，让人笑一下也是善意，也是态度，也是传播。

---

## 二、情绪内核的命名方法论

**情绪内核 = 一个具体的情感时刻，不是抽象风格词。**

为什么必须是具体时刻？因为抽象形容词（「孤独」「治愈」「温暖」）是**结论**，它已经把感受压扁了；而具体时刻是**现场**，它让人重新经历那个感受，从而被击中。传播来自「被重新经历」，不来自「被告知一个结论」。

**质量标准：**

| ❌ 抽象形容词 | ✅ 具体情感时刻 |
|------------|---------------|
| 孤独感 | 一个人深夜刷手机，突然不知道自己在等什么 |
| 成就感 | 做完一件很难的事，没人知道，自己给自己鼓掌的那一秒 |
| 治愈 | 像有人轻轻拍了一下你的肩膀，但没有说话 |
| 搞笑 | 在地铁上看到某个东西，嘴角忍不住往上，旁边有人，硬压下去 |
| 荒诞 | 用自己的脸对上名画，发现长得一样，突然觉得所谓高雅很好笑 |
| 反差 | 认真做了一件很蠢的事，但做完之后感觉良好 |

**沉的不比轻的深刻，快乐的不比悲伤的低级。** 内核的质量取决于它是否具体、真实，不是它的重量。

## 三、20 个好的情绪内核示例（覆盖完整光谱）

### 沉的（5）
1. 深夜把想说的话打出来又删掉，最后只发了一个句号
2. 做完一件很难的事，没人知道，自己给自己鼓掌的那一秒
3. 走在回家的路上，突然不想到家，又多绕了两条街
4. 翻到三年前的聊天记录，发现那时的自己什么都不怕
5. 把一句早就想说的「谢谢」，最终还是没发出去

### 中的（5）
6. 总有一个地方现在是下午五点钟，那里的光和你这里不一样
7. 专注的时候，时间不是被消耗，是在飞向某个地方
8. 把杂乱的脑子里那团东西，第一次看见它长什么样
9. 终于有人没让我「改掉」我的毛病，而是说这是另一种操作系统
10. 一个人吃饭时，把手机立起来假装在跟谁视频

### 轻的（5）
11. 打开之后嘴角压不住，但又不知道该怎么跟别人解释这个东西
12. 想发给某个人说「你看这个」，但又怕对方觉得无聊
13. 屏幕上只有一个表情，你越想忍它越抖
14. 做了一件很蠢的事，但做完感觉良好
15. 截图发到群里，所有人都回了同一个表情

### 荒诞 / 反差的（5）
16. 每两天按一次「我还活着」，按完突然觉得活着这件事需要被确认
17. 你必须先去摸真实的草，才能解锁摸虚拟的世界
18. 测出来自己是「吗喽」，居然有点骄傲地转发了
19. 用「死亡检测」这种医疗概念，来确认自己今天还在
20. 把 MBTI 的严肃，错位成一场所有人共谋的自嘲

## 四、从模糊描述到精准命名（转化示例）

| 用户模糊描述 | ❌ 错误命名（太抽象/被升华） | ✅ 精准命名 |
|------------|------------------------|-----------|
| 「想做个治愈的东西」 | 温暖治愈的陪伴感 | 像有人轻轻拍了下你肩膀，但没说话 |
| 「一个让人哈哈哈的小工具」 | 其实背后有一种轻盈的对抗 | 屏幕上只有一个表情，你越忍它越抖 |
| 「确认自己还活着的 app」 | 关于生命意义的深刻叩问 | 每两天按一次「我还活着」，按完发现活着需要被确认 |
| 「戒手机的应用」 | 数字时代的自我救赎 | 你必须先摸真实的草，才能解锁摸虚拟世界 |

## 五、反升华警示（高频错误）

「升华」= 把一个轻的/荒诞的/好笑的 idea，强行拉向沉重、诗意、深刻。这是本技能最常见、最该避免的错误。

| 用户的 idea | ❌ 升华（禁止） | ✅ 保留本来的样子 |
|-----------|--------------|----------------|
| 想做让人笑的东西 | 「其实背后有一种轻盈的对抗……」 | 「打开就想笑，然后马上想发给某个人」 |
| 自嘲标签测试 | 「这是当代年轻人的精神困境写照」 | 「测出来是吗喽，但骄傲地转发了」 |
| 摸草解锁手机 | 「一场关于人与自然关系的哲学实践」 | 「我承认我上瘾了，但我用一种很蠢的方式对抗」 |
| 抖动的表情 | 「关于人类情感压抑的静默体验」 | 「你越想忍它越抖，最后没忍住」 |

**判断方法**：如果你写的句子里出现「其实背后」「关于……的深刻」「一种诗意的」，而用户的 idea 本来是轻的——停下，删掉，用轻的语言重写。

## 六、B 轴 · 六种传播动力

传播动力回答「为什么有人会转发」——心理动力是什么。Phase 3 每个发散方向必须声明它依赖哪种动力。

| 传播动力 | 用户心理 | 典型传播动作 | 案例参照 |
|---------|---------|------------|---------|
| **被代言感** | 「这说的就是我」 | 转发配文「就是我」、收藏 | Eternal Sunset、思花、专注飞机 |
| **参与冲动** | 「我也要试一下」（含「行动即内容」） | 自己做一个版本发出去 | 名画对脸、Touch Grass |
| **反差击中** | 「我没想到」 | 转发配文「神了」 | 死了么、名画对脸 |
| **压不住的笑** | 「嘴角忍不住」 | 转发配文「哈哈哈哈哈」 | （待补充案例） |
| **荒诞共谋** | 「？？？但我看了三遍」 | 转发配文「？？？」「这也太对了」 | SBTI、Touch Grass |
| **身份归属** | 「这是为我做的 / 我们是同类」 | 关注、深度使用、自我标签化 | PlanCoach、SBTI |

### 每种动力的展开

- **被代言感**：产品替用户说出了他说不出/不好意思说的话。传播动作是「转发即认领」。关键：要精准到「就是我」，泛泛的共鸣不够。
- **参与冲动**：产品让用户完成一个动作，动作本身就是内容（「行动即内容」）。用户做完会想晒出「我的版本」。关键：动作要低门槛 + 有个性化结果。
  - 注：旧版本的「参与证明型」其实就是这种动力的产品形态表现，已归并到这里。
- **反差击中**：功能/形式与它问的问题之间有巨大落差（死了么：动作极简单，问题极沉重）。落差本身就是内容。
- **压不住的笑**：纯粹好笑，不需要意义。传播动作是「哈哈哈哈哈」+ 转给朋友。不要给它加深度。
- **荒诞共谋**：「？？？但我看了三遍」——一种「大家一起承认某件荒诞的事」的快乐。SBTI 的「我们都是吗喽」就是共谋。
- **身份归属**：「这是为我这种人做的」。用户用它来标记自己属于某个群体。传播动作偏深度（关注、长期使用、自我标签化）。

### 动力 × 情绪重量的关系

传播动力和情绪重量**正交**——任何重量的情绪都可以走任何动力。但有些组合更自然：
- 沉/中 的情绪常走 被代言感 / 身份归属
- 轻/荒诞 的情绪常走 压不住的笑 / 荒诞共谋 / 反差击中
- 参与冲动 几乎任何重量都适用（关键在动作设计）

这不是限制，只是起点。刻意打破常见组合（如给沉的情绪配荒诞共谋）往往能产出意外的好方向。
