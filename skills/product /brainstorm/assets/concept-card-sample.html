过劳报警器

OverWork Alarm

一个让你越努力它越受不了、会当场抓你现行的报警器。

情绪内核
被进度条追了一整天，一个东西突然一本正经命令你「快去浪费时间」，你愣一下然后偷偷笑了。
思维方法
自由漂移（反转 + 极端化：把「提醒你去浪费」反过来——监测过度高效，推到最极端的对抗）
感知形态
身体型 + 声音型
传播动力
压不住的笑

候选产品名

过劳报警器 OverWork Alarm 太努力了 警告

核心情绪时刻

正干得起劲，机器突然像抓现行一样「滴——滴——」红灯狂闪，屏幕黑掉甩出一句「检测到过劳，已为您强制休息」——你被气笑了，但手真的停了下来。

交互隐喻

持续高效工作 = 触发警报，被强制摸鱼

视觉方向

安全警示黄 + 警报红 + 工业黑，像消防报警器或施工警示牌。进度条悄悄涨满后，整屏「红黑频闪」+ 横幅式警告滑入，像系统蓝屏那样毫无商量——一本正经的工业级严肃，配上荒唐的报警理由，反差越硬越好笑。

传播钩子

  • 全网第一个你越努力它越生气的 App
  • 检测到过劳，已为您强制休息。
  • 别人的闹钟叫你起床，我的闹钟叫我别干了。

Vibe Coding 提示词

过劳报警器（OverWork Alarm）交互原型

视觉风格：industrial warning, hazard yellow & alarm red, blinking siren, system-alert banner, monospaced
核心动效：后台计时连续高效时长，进度满后整屏红黑频闪 + 顶部滑入警告横幅 + 短促「滴滴」蜂鸣，屏幕变灰锁定 N 分钟倒计时
技术方向：Web 前端（React + CSS keyframes 频闪 + Web Audio API 蜂鸣）
情绪氛围：deadpan-serious, absurd, laugh-out-loud
核心交互逻辑：用户开始工作即计时；连续高效达阈值（如 90 分钟无中断）触发全屏报警，强制锁屏进入「摸鱼倒计时」。

请构建一个可在浏览器中运行的交互原型，重点实现「连续高效 → 全屏红黑频闪报警 → 强制摸鱼锁屏倒计时」，整体风格参考工业警示牌，让用户感受到「正干得起劲被机器抓现行、气笑了但真停下来」。
