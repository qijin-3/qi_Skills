# Monthly Goal · 输出模板

## Step 2 · 上月总结

```
📅 {YYYY} 年 {MM-1} 月复盘

✅ 完成任务共 {N} 条（我负责的，上月完成）
  Writing:{n} | Growing:{n} | Projects:{n} | Routine:{n} | Idea:{n}

📋 今天要做平均完成率：{X}%
   {如有连续低完成率周：提示「第 N 周曾连续低效」}
   {若日志数据不足：注明「日志样本不足，以下结合周复盘推断」}

🎯 上月目标：
   1. 「{目标1}」→ 进行中（待打分）
   2. 「{目标2}」→ 进行中（待打分）

🔄 分组博弈高频模式：{模式摘要，无 snapshot 则写「无快照数据」}

{能量信号摘要（可选，≤2 句）}
```

## Step 3 · 打分提议

```
📝 「{目标标题}」打分建议：{X}%

依据：{完成情况 + 验收标准对比，1-2 句}

确认？（或输入你的分数）
```

## Step 4a · 本月目标草案

```
🚀 本月目标建议（{YYYY}-{MM}）

1. [{领域}] {目标描述}
   验收标准：{具体可量化}
   理由：{1 句}

2. [{领域}] {目标描述}
   验收标准：{具体可量化}
   理由：{1 句}

刻意不做：{这个月主动搁置的方向}

这个月预计有效工作日多少天？（去掉旅行/密集社交等）
```

## Step 4b · 可行性校验

```
{N} 个工作日，目标 1 预计需要 ~X 小时/周，可行。
目标 2 加进来后总体偏紧，建议把「{某部分}」推到下月？
```

## Step 5 · 周主线

```
📆 本月周主线拆解：

W{nn}（{MM-DD}~{MM-DD}）：{主线1}
W{nn+1}：{主线2}
W{nn+2}：{主线3}
W{nn+3}（{MM-DD}~）：{主线4}
{W5 若月末有余天：W{nn+4}（{MM-DD}~{MM-DD}）：月末收尾}

大致合理吗？
```

## Step 7 · 确认消息

```
✅ {YYYY}-{MM} 月度计划已生成

目标 {N} 条已写入目标表
月计划已保存：content/{YYYY}/{MM}/monthly-plan.md

上月目标已打分并归档。

本月第一周主线：「{W1 主线}」
祝这个月顺利 💪
```
