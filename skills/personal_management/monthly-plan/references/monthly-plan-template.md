# Monthly Goal · 本地月计划模板

写入路径：`Personal-os/content/{YYYY}/{MM}/monthly-plan.md`

```markdown
# {YYYY}-{MM} 月度计划

## 本月可用时间（写实）
- 有效工作日约 {N} 天；已知占用：{用户提到的}

## 本月目标（{N} 条，每条可验证）
1. [{领域}] {目标描述} — 验收标准：{标准}
2. ...

## 刻意不做（防止贪多）
- {内容}

## 拆成周主线
- W{nn}（{MM-DD}~{MM-DD}）：{主线1}
- W{nn+1}：{主线2}
- ...

## 与上月对比
- 上月任务完成 {N} 条；今天要做平均完成率 {X}%
- 目标完成度：{目标1}={X}% / {目标2}={X}%
- 被饿死的目标：{若有}

## 本月子文档（同目录）
- `weekly-review.md` — 周复盘（一月一文件，按周追加）
- `diary.md` — 日记（一月一文件，按日追加）
- `about-me-updates.md` — About Me 更新记录（变更前先写，仅本地）
```

## 同目录初始化文件

若不存在则创建，仅写标题行：

| 文件 | 首行 |
|------|------|
| `weekly-review.md` | `# {YYYY}-{MM} 周复盘` |
| `diary.md` | `# {YYYY}-{MM} 日记` |
| `about-me-updates.md` | `# {YYYY}-{MM} 关于我 · 更新记录` |
