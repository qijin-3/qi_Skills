# 电影风格参考库

每种风格包含：触发关键词（用于第 2 步提问推导）、CSS 变量值、地图底图选择、文字调性指引。

## 旅途基调 → 导演推荐映射

| 用户说的旅途类型 | 推荐导演/作品 | 对应预设 |
|---|---|---|
| 史诗公路、辽阔、自由 | 文德斯《公路之王》、科恩兄弟 | 风格 1（黄金公路）|
| 宁静、留白、日常诗意 | 是枝裕和、小津安二郎 | 风格 2（冰雪禅意）|
| 精神圣地、仪式感、历史 | 贝托鲁奇《末代皇帝》、塔可夫斯基 | 风格 4（圣地金红）|
| 壮阔自然、敬畏、纪录 | 维尔纳·赫尔佐格、BBC | 风格 3（大地翠绿）|
| 城市漫游、偶遇、轻盈 | 法国新浪潮、特吕弗 | 风格 5（浪漫薄紫）|
| 极简、疏离、现代感 | 安东尼奥尼、早期戈达尔 | 风格 6（极简黑白）|

## 风格 1：黄金公路（公路史诗）

**电影参考**：《末路狂花》《荒野生存》《公路》文德斯风格  
**情绪**：自由、辽阔、荒凉中的温暖、公路诗意

### CSS 变量
```css
--ink: #09070a;
--ink-soft: #14101a;
--paper: #e8d4a0;
--paper-dim: rgba(232, 212, 160, 0.72);
--line: rgba(232, 212, 160, 0.2);
--line-strong: rgba(232, 212, 160, 0.42);
--red: #c45a2a;
--blue: #5a7a9a;
--white: #ddc898;
--green: #5a7040;
--blackGold: #c4883c;
--tone: var(--blackGold);
--tone-soft: rgba(196, 136, 60, 0.18);
```

### 地图底图
`activeStyle: "cinematic-dark"`（暗色地图，配合金色路线）

### 路线颜色
`routes.full.color: "rgba(232, 212, 160, 0.16)"`  
`routes.active` 默认即可

### 文字调性
句式宽松，留有停顿，带公路诗意。多用简短单句和破折号。强调空间感和移动感，如「风从西来，路没有尽头」。避免过度抒情，保持粗粝的真实。

## 风格 2：冰雪禅意（日式克制）

**电影参考**：是枝裕和、北野武、小津安二郎  
**情绪**：留白、寂静、内敛、日常中的诗意

### CSS 变量
```css
--ink: #0c0c0e;
--ink-soft: #181820;
--paper: #e9e8e4;
--paper-dim: rgba(233, 232, 228, 0.72);
--line: rgba(233, 232, 228, 0.18);
--line-strong: rgba(233, 232, 228, 0.38);
--red: #a04040;
--blue: #4a6880;
--white: #d8d8d4;
--green: #4a6050;
--blackGold: #7a8fa6;
--tone: var(--blackGold);
--tone-soft: rgba(122, 143, 166, 0.18);
```

### 地图底图
`activeStyle: "paper-muted"`（浅灰地图，配合冷色调）

### 路线颜色
`routes.full.color: "rgba(233, 232, 228, 0.14)"`

### 文字调性
短句居多，克制收敛。不直接描写情绪，通过细节暗示感受。如「早饭是白粥。窗外下雪了。」多用白描，少用形容词，适度空行制造留白感。

## 风格 3：大地翠绿（自然纪录片）

**电影参考**：BBC 蓝色星球、《人类星球》、维尔纳·赫尔佐格纪录片  
**情绪**：壮阔、敬畏、生命力、自然与人的渺小

### CSS 变量
```css
--ink: #060d09;
--ink-soft: #0f1a10;
--paper: #dcecd6;
--paper-dim: rgba(220, 236, 214, 0.72);
--line: rgba(220, 236, 214, 0.18);
--line-strong: rgba(220, 236, 214, 0.38);
--red: #8a4030;
--blue: #2a5870;
--white: #c8d8c0;
--green: #3d7a5e;
--blackGold: #4a8a5a;
--tone: var(--blackGold);
--tone-soft: rgba(74, 138, 90, 0.18);
```

### 地图底图
`activeStyle: "cinematic-warm"`（暖色自然地图）

### 路线颜色
`routes.full.color: "rgba(220, 236, 214, 0.14)"`

### 文字调性
描述性强，注重感官细节（气味、温度、声音）。可以用数字和精确的自然描述，如「海拔 4580 米，气温零下三度，风速让帽檐不停翻飞」。叙述角度带有观察者的距离感，像是在记录一个比自己更大的事物。

## 风格 4：圣地金红（贝托鲁奇·史诗宗教）

**电影参考**：贝托鲁奇《末代皇帝》《遮蔽的天空》、塔可夫斯基  
**情绪**：仪式感、厚重、历史与精神的交叠、沉默中的壮美

### CSS 变量
```css
--ink: #0d0b08;
--ink-soft: #18120d;
--paper: #f4ead8;
--paper-dim: rgba(244, 234, 216, 0.72);
--line: rgba(247, 240, 223, 0.2);
--line-strong: rgba(247, 240, 223, 0.42);
--red: #b73525;
--blue: #2d5f8d;
--white: #e9dfcc;
--green: #3f6f4a;
--blackGold: #b88a3b;
--tone: var(--blackGold);
--tone-soft: rgba(184, 138, 59, 0.18);
```

### 地图底图
`activeStyle: "cinematic-dark"`

### 路线颜色（默认）
`routes.full.color: "rgba(244, 234, 216, 0.16)"`

### 文字调性
文言与白话交织，句子有分量。用传统的空间词汇（「谒」「抵」「驻」「望中」），但不脱离现代语感。每段话有重心，不平铺直叙。带有历史厚重感，但保持个人视角的真实温度。

## 风格 5：浪漫薄紫（法国新浪潮）

**电影参考**：特吕弗、里维特、香特尔·阿克曼、《戏梦巴黎》  
**情绪**：自由漂浮、哲思、偶遇、年轻的忧郁

### CSS 变量

```css
--ink-soft: #1e1a2a;
--paper: #f0ecf8;

--line-strong: rgba(240, 236, 248, 0.38);
--red: #9a4060;
--blue: #4050a0;
--white: #e0dcea;
--green: #506070;
--blackGold: #8b6fa8;
--tone: var(--blackGold);
--tone-soft: rgba(139, 111, 168, 0.18);
```

### 地图底图
`activeStyle: "paper-muted"`

### 路线颜色
`routes.full.color: "rgba(240, 236, 248, 0.14)"`

### 文字调性
带有碎片感和跳跃性，可以有未完成的句子和疑问。语气轻盈，不沉重。对话和细节多，有时会突然插入一个哲学式思考，然后又跳回来描述路边的一碗米饭。

## 风格 6：极简黑白（安东尼奥尼·现代）

**电影参考**：安东尼奥尼、戈达尔黑白时期、《精疲力竭》  
**情绪**：疏离、都市感、冷静凝视、存在主义

### CSS 变量
```css
--ink: #080808;
--ink-soft: #141414;
--paper: #f0f0f0;
--paper-dim: rgba(240, 240, 240, 0.72);
--line: rgba(240, 240, 240, 0.16);
--line-strong: rgba(240, 240, 240, 0.36);
--red: #cc3333;
--blue: #336699;
--white: #d8d8d8;
--green: #447744;
--blackGold: #888888;
--tone: var(--blackGold);
--tone-soft: rgba(136, 136, 136, 0.18);
```

### 地图底图
`activeStyle: "paper-muted"`

### 路线颜色
`routes.full.color: "rgba(240, 240, 240, 0.12)"`

### 文字调性
极度克制，几乎没有形容词。叙述事实，让读者自己感受。句子短而精准。可以有意留下未解释的细节，制造疏离感。如「到了。住进去。房间很白。」不解释为什么，也不提供情绪。

## 自定义风格（用户描述）

当用户描述自己想要的风格时（如「我想要那种日落橘红的感觉」「像九十年代香港电影」），根据描述即兴生成一套 CSS 变量，参照以上格式。

生成原则：
- `--ink`：最深背景色，通常是偏色的深黑
- `--paper`：主要文字/前景色，带有风格倾向的暖/冷/中性调
- `--blackGold`（实际上是 `--tone`）：强调色，最能体现风格的标志色
- `--red/blue/green/white`：辅助色，用于各天的「tone」标记
- 颜色之间要有足够的对比度，保证可读性
