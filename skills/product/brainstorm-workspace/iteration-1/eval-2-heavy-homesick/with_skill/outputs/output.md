▶ Phase 1 · 情绪解码

**1a · 先说感受**

这个 idea 给我的第一感觉是——一个人在出租屋里跟家里视频，特意把镜头往后退了半步，好让背景看起来没那么空；嘴上说「这边都挺好的，公司管饭」，挂掉之后才发现自己一整天没吃东西。有点像「报喜不报忧」那四个字被你活成了一种日常技术活的那种感觉。

它不是孤独，孤独是没人，而你这里是有人的——电话那头有人，只是你不能让他们看见真的你。它也不是惨，你没在卖惨，你是在**用力维持一个『我很好』的版本**，而且维持得越好，越没人知道你维持得有多累。

这个方向对吗？还是你想做的那个「假装」，其实没这么沉，更接近一种心照不宣的、大家都在演的小默契？

**1b · 定向问题（本轮 2 个）**

- **目标情绪**：你希望刷到它的人，是「鼻子一酸然后想转给某个也在外地的人」，还是「会心一笑——因为他也这么演过」？这两个会把产品带向完全不同的地方。
- **态度立场**：这个 idea 在替谁说话？是替「报喜不报忧的孩子」喊一句「我也想被问问累不累」，还是替「被瞒着的父母」说一句「我其实都看得出来」？它站哪一边，决定了它的后劲。

**1c · 命名情绪内核并确认**

（假设用户确认：目标情绪 = 鼻子一酸 + 想转给同样在外地的人；立场 = 替「报喜不报忧的孩子」说话，但留一个口子让父母那一端也能被看见。情绪重量确认为「沉 / 中」，不要往「图一乐」的方向走，也不要强行升华成宏大乡愁。）

> 情绪内核：跟家里视频时把「我很好」说得特别熟练，挂掉那一秒，房间一下子安静下来

呈现：我把这个 idea 的情绪内核理解成上面这句——重点不在「想家」，在「**那个被你练熟的『我很好』，和挂断后的安静之间的落差**」。准确吗？

（假设用户确认：准确，就锁这个内核进入下一阶段。）

▶ Phase 3 · 情绪地图发散

把同一个内核——「『我很好』说得特别熟练，挂断后的安静」——发散成 8 个方向。刻意混用 5 种思维方法，其中含 2 个自由漂移、2 个概念错位，确保不全是案例库变体。语言重量与内核对齐（沉 / 中），不往荒诞走。

---

【方向一 · 我很好生成器】
思维方法：触发反推（先锁定「想转给同样在外地的人」+「就是我」，再反推出一个让人一键认领的语言型产品）
双轴定位：感知形态[语言型] × 传播动力[被代言感]
交互隐喻：输入今天真实的烂事 → 它帮你翻译成一句能发给爸妈的「我很好」 = 把你每天做的那个翻译动作，明明白白摆出来
视觉意象：上下两个对话框。上面灰色小字是你打的真话「今天被领导骂了，外卖洒了一半」，按一下，下面浮出一句温柔的蓝色字「妈我挺好的，今天还加了个鸡腿」，中间一条细线像一道没说出口的坎
情绪时刻：打开第一秒你会笑一下，因为太准了；笑完有点想哭，因为你天天在干这件事，只是从没见过它被写出来

【方向二 · 挂断后的 30 秒】
思维方法：形态映射（把内核里「挂掉那一秒的安静」直接映射成时间型——产品只发生在通话结束之后）
双轴定位：感知形态[时间型 + 声音型] × 传播动力[被代言感]
交互隐喻：录下你刚跟家里说完「我很好」挂断后的那 30 秒环境音 = 把那段没人听见的安静，留个证据
视觉意象：纯黑屏，中间一条极细的声波线，安静时几乎是直的，偶尔抖一下（你叹了口气、咽了口水、椅子响了一声）；30 秒结束，屏幕底部慢慢浮出一行小字「这 30 秒，只有你知道」
情绪时刻：你会发现那 30 秒长得不可思议，安静本身原来是有重量的

【方向三 · 报平安·年度对账单】
思维方法：概念错位（把「银行年度账单 / 财务对账」这种冷冰冰的官方流程，搬到「你跟家里报的平安」上）
双轴定位：感知形态[视觉型 + 语言型] × 传播动力[反差击中]
交互隐喻：年底生成一张「平安对账单」，列出你这一年报了多少次「我很好」、瞒下了多少件事 = 用最不近人情的报表格式，算一笔最走心的账
视觉意象：一张仿真的银行流水单 / 账单，宋体黑字、表格线、印章。条目却是「全年报喜 142 次 / 实际加班 96 天 / 隐瞒生病 5 次 / 视频时退后半步 31 次」，末尾盖一个红章「本年度平安，已对平」
情绪时刻：前几秒你以为是个搞笑模板，看到「隐瞒生病 5 次」那行，突然就笑不出来了——反差就在这里

【方向四 · 背景里没有的那部分】
思维方法：概念错位（把「视频通话虚拟背景 / 美颜滤镜」这个我们天天用的功能，反过来用——它不美化，它专门暴露你藏在镜头外的东西）
双轴定位：感知形态[视觉型 + 空间型] × 传播动力[被代言感]
交互隐喻：上传一张你视频时的「对外背景」，它帮你画出镜头框外真实的那一圈 = 把你特意退后半步、特意避开的那部分房间，补全给你自己看
视觉意象：屏幕中间一个亮着的视频画面（整洁的一角、绿植、暖光），画面四周是暗下去的、手绘风的延伸——堆着的快递箱、没洗的碗、晾了三天的衣服、地上的药盒。镜头内外一明一暗
情绪时刻：你盯着那个亮亮的小框，知道家里人只看得到这一块，而你住在剩下那一大片暗里

【方向五 · 一句话回声墙】
思维方法：案例锚定（参考「思花」把说不出口的情绪可视化、被代言的手法，做成一面公共的、匿名的留言场）
双轴定位：感知形态[视觉型 + 关系型] × 传播动力[被代言感 + 身份归属]
交互隐喻：写下一句你今年对家里撒过的「善意的谎」，它飘进一面所有人的墙 = 你发现撒这种谎的不止你一个，整面墙都是
视觉意象：深夜蓝的一面墙，无数张小纸条般的句子缓慢上浮又下沉，像水里的灯。「我说我胖了，其实是浮肿」「我说同事对我很好，其实没人记得我生日」。你写的那一条，会亮一下，然后汇进去
情绪时刻：你以为自己在一个人演,抬头发现满墙都是同一场戏的演员

【方向六 · 我替你接一次】
思维方法：触发反推（从「想替对方说出口」这个传播动作反推，做成一个关系型的角色互换器）
双轴定位：感知形态[关系型 + 语言型] × 传播动力[被代言感]
交互隐喻：切换成「爸妈视角」，让你以父母的身份，给「报喜不报忧的自己」回一条消息 = 你被迫站到桌子对面，去想他们其实想说什么
视觉意象：聊天界面突然左右对调，你的头像移到了「爸妈」那一栏。输入框上方有句灰字提示「现在，你是妈妈。你会回他什么？」你打字的手会停很久
情绪时刻：当你以妈妈的身份打出「我看得出来你没说实话，但我不戳穿你」时，你才听见那句你一直希望听到的话——是你自己说给自己的

【方向七 · 时差里的那盏灯】（自由漂移 · 随机词碰撞「电表」）
思维方法：自由漂移（随机抓「电表 / 用电曲线」这个跟情绪完全无关的物件，强行和内核相撞，不参照任何案例）
双轴定位：感知形态[时间型 + 空间型] × 传播动力[被代言感]
交互隐喻：把你的「深夜还醒着」可视化成一条用电曲线，和家乡那座城市的「集体熄灯时间」叠在一起 = 用一条线证明，你的城市在你还亮着的时候，家已经睡了
视觉意象：一张极简的折线图，横轴是 24 小时。一条暖色线（家乡）早早归零熄灭，一条冷色线（你）在凌晨还高高翘着。两条线之间那块空白区域，被标注成「你说『早睡了』的那几个小时」
情绪时刻：你看着那两条永远错开的线，第一次具体地、用数据地理解了「时差」其实是「我醒着的时候，能担心我的人都睡了」

【方向八 · 把『没事』存进罐子】（自由漂移 · 极端化 + 反转）
思维方法：自由漂移（极端化：把「报喜不报忧」推到最极端——一整年所有的『没事』都不丢，全存起来；反转：别人做记录是为了回看美好，这个是为了有一天能被打开看见真相）
双轴定位：感知形态[身体型 + 时间型] × 传播动力[被代言感]
交互隐喻：每说一次「没事 / 我很好」，就往一个虚拟罐子里投一颗石子 = 把你咽下去的话，攒成一个有重量、看得见的东西
视觉意象：一个透明玻璃罐，每投一颗石子有一声很轻的「叮」。罐子慢慢满，越满越沉，底部的石子开始反光。年底可以选择「倒出来」——所有石子哗啦落下，每一颗浮出当时你真正想说的那句话
情绪时刻：平时只是「叮」一声很轻松，直到某天你看见罐子快满了，才意识到原来咽下去了这么多

---

质量自检：8 个方向均声明了双轴 + 思维方法；含 2 个概念错位（三、四）+ 2 个自由漂移（七、八）+ 形态映射 / 触发反推 / 案例锚定各有覆盖；全程语言保持「沉 / 中」，没有把它做成搞笑模板，也没有强行升华成宏大乡愁（对账单方向虽有反差感，但落点仍是真实的隐瞒，不是图一乐）。

收尾提问：哪 1–2 个方向最让你有感觉？

（假设用户确认：最有感觉的是 **方向一「我很好生成器」**（最直接、最像「就是我」、传播门槛最低）和 **方向三「报平安·年度对账单」**（反差强、天然可截图、适合年底节点）。下面对这两个出完整概念卡。基于传播动力，我也推荐这两个：一个走纯被代言感、低门槛即时认领，一个走反差击中、强截图属性，组合互补。）

▶ Phase 4 · 交互原型概念卡

================== 概念卡 ① ==================

**字段一 · 产品名 + 一句话定义**
候选名：
1. **我很好生成器**（最直白，搜索友好）
2. **报喜机**（短，有点旧式电报机的温度）
3. **挺好的**（口语，就是那句话本身）

一句话定义：一个**让你把今天的烂事，翻译成一句能发给爸妈的「我很好」的**小工具。

**字段二 · 核心情绪时刻**
打开第一秒你会笑一下，因为太准了——它居然知道你天天在干这件事；笑完那口气没松下去，因为你忽然看见了自己每天做的那个无声的翻译动作，被白纸黑字写了出来。

**字段三 · 交互隐喻**
把真话打进去、换出一句假的好 = 把你每天替家人做的那次「情绪美颜」，第一次摆到台面上看清楚。
（用户动作：输入真实的糟糕 → 生成善意的「我很好」 = 情绪意义：原来「报喜不报忧」是一门你早就练熟的手艺）

**字段四 · 视觉方向**
- 色彩基调：旧纸米白 + 真话用的冷灰 + 假话「我很好」用的暖蓝（像家里那盏灯的颜色）
- 核心视觉动作：你打下的灰色真话，按下按钮后，字一个一个被「擦掉」，从底下慢慢浮出暖蓝色的那句假话；中间始终有一条没说破的细线把上下隔开
- 情绪氛围：克制、温柔、像深夜一个人打字时屏幕的光——安静，但喉咙有点紧

**字段五 · Vibe Coding 提示词**
```
我很好生成器 交互原型

视觉风格：minimal, warm-paper texture, two-tone (cold-gray truth / warm-blue lie), serif for the "lie", quiet
核心动效：用户在上方输入框打入真话（冷灰色）；点击「翻译成我很好」后，灰字逐字 dissolve / erase，下方暖蓝色的善意版本逐字 typewriter 浮现；上下两栏之间始终保留一条 1px 的细线，hover 时浮出一句小字「这是你没说的那部分」
技术方向：纯前端即可，HTML + CSS Animation + 一点 JS 控制逐字动画；翻译逻辑可先用规则模板（情绪关键词 → 善意改写），后续可接 LLM API
情绪氛围：tender, restrained, bittersweet
核心交互逻辑：单页。① 用户输入今天真实发生的糟糕事；② 点击生成，得到一句可直接复制、发给家人的「我很好」版本；③ 生成结果下方有「保存这条没说出口的真话」入口（攒成私人记录，呼应「罐子」概念）；④ 可一键生成「真话 / 假话」对照图，便于截图分享。
请构建一个可在浏览器中运行的交互原型，重点实现「灰字被擦掉、暖蓝字浮现」的逐字转换动效，整体风格参考 minimal warm-paper / two-tone，让用户感受到「太准了，笑完又有点想哭」。
```

**字段六 · 传播钩子**（给 3 个候选）
1. 「我没在报喜，我在替你们把忧藏好。」
2. 「『我很好』这三个字，我已经熟练到不用过脑子了。」
3. 「输入真话，它帮你翻译成爸妈能睡得着的版本。」

================== 概念卡 ② ==================

**字段一 · 产品名 + 一句话定义**
候选名：
1. **平安对账单**（概念错位最清晰）
2. **报喜流水**（仿银行流水，反差更冷）
3. **年度报平安**（最易懂）

一句话定义：一个**让你看见自己这一年到底报了多少次「我很好」、瞒了多少件事的**年终账单。

**字段二 · 核心情绪时刻**
头几秒你以为这又是个年终总结的搞笑模板，准备截图玩玩；当目光扫到「隐瞒生病 5 次」「视频时退后半步 31 次」那两行，手指停住了——这账，原来真的对得平。

**字段三 · 交互隐喻**
把一年的「报喜不报忧」做成一张冷冰冰的对账单 = 用最不近人情的报表格式，逼自己正视一笔最走心的账。
（用户动作：生成并翻看自己的平安账单 = 情绪意义：那些随口说的「没事」，被一条条记下来后，原来有这么重）

**字段四 · 视觉方向**
- 色彩基调：账单白 + 宋体墨黑 + 一枚印泥红的章
- 核心视觉动作：条目像打印机一样一行行「哒哒哒」滚动打出来；最后一行打完，一枚红章从上方「啪」地盖下「本年度平安，已对平」，盖章瞬间纸面轻微一震
- 情绪氛围：表面公事公办、冷静克制，内里越往下看越沉——像在看一份关于自己的、太诚实的报告

**字段五 · Vibe Coding 提示词**
```
平安对账单 交互原型

视觉风格：bank-statement / receipt aesthetic, serif (宋体), monochrome ink on paper, one red seal stamp, formal
核心动效：条目逐行 print-out（打印机滚动 + 轻微抖动 + 打字音效感）；全部打完后红色印章 scale-in + 落章微震；提供「下载 / 长图截图」按钮
技术方向：纯前端，HTML + CSS + JS；条目数据可由用户简单几步自评输入（滑块/计数）或预置模板生成；导出长图用 html2canvas
情绪氛围：deadpan, formal-on-the-surface, quietly heavy
核心交互逻辑：① 用户通过几个轻量问题自评（今年报喜约几次、瞒过几次生病、视频时是否退后过…）；② 系统生成一张仿真「平安对账单」，含条目、合计、红章；③ 一键导出竖版长图，适合发朋友圈/小红书；④ 末尾留一行可编辑的「附言」，让用户给家里写一句这次想说的真话。
请构建一个可在浏览器中运行的交互原型，重点实现「逐行打印 + 落章微震」的动效，整体风格参考真实银行对账单 / 收据，让用户感受到「以为是搞笑模板，看到某一行突然笑不出来」的反差。
```

**字段六 · 传播钩子**（给 3 个候选）
1. 「全年报喜 142 次，实际加班 96 天——本年度平安，已对平。」
2. 「这是我给爸妈的年度报告，删掉的那部分才是真的。」
3. 「报喜不报忧，原来也是要对账的。」
