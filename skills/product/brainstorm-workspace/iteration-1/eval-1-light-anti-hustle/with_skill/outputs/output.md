▶ Phase 1 · 情绪解码

这个 idea 给我的第一感觉是——一个被待办、打卡、进度条追了一整天的人，手机突然一本正经地弹出来命令他「你今天还没浪费时间，快去浪费一点」，他愣了半秒，然后偷偷笑了出来。有点像上班摸鱼时突然收到一张盖了红章的「官方摸鱼许可证」的那种感觉：明明是放纵，却被一个比你还严肃的系统下令执行，于是放纵这件事突然变得理直气壮。

我特意没往「现代人好累、好治愈、要善待自己」那个方向走——你这个 idea 的好笑恰恰在于它**用卷的语气劝你别卷**，是个反差梗，不是一碗鸡汤。我想守住这股「一本正经地胡闹」的劲儿。

（按 skill 规则，单轮不超过 2 个问题，我先问两个最定向的：）

1. **目标情绪**：你希望看到的人是「噗嗤一声笑出来，然后截图发给那个最卷的朋友」，还是「会心一笑，默默自己用」？这决定它是往「压不住的笑」还是「被代言感」走。
2. **态度立场**：它在对抗什么？是对抗「打卡 App / 时间管理 / 效率工具」这一整套话术本身（用它们的语气反过来玩它们），还是单纯想给人一个放松的借口？

> （假设用户确认：希望别人「噗嗤笑出来想转发」，并且就是要**对抗效率工具那套话术本身**——用打卡 App 的语气，反过来逼你摸鱼。这把方向锁定在「荒诞共谋 / 压不住的笑」，而非温情的「被代言感」。）

**1c · 命名情绪内核**

情绪内核：**被进度条追了一整天，一个东西突然一本正经命令你「快去浪费时间」，你愣一下然后偷偷笑了。**

我把这个 idea 的情绪内核理解为上面这句。它是轻的、带反差的，核心快感来自「严肃的形式 × 胡闹的内容」这个错位——不是「关爱自己」的暖，是「被一个比我还轴的系统下令偷懒」的好笑。准确吗？

> （假设用户确认：准确，就是这个味，别给我加温情滤镜。进入下一阶段。）

▶ Phase 3 · 情绪地图发散

把同一个情绪内核——「被一个一本正经的系统下令去浪费时间，愣一下然后偷偷笑」——发散成 8 个方向。守住「严肃形式 × 胡闹内容」的反差，不升华。

【方向 1 · 浪费时间 KPI】
思维方法：概念错位（把职场最让人窒息的「KPI / 绩效考核」从公司语境，搬到「逼你摸鱼」的反语境，看它变成什么）
双轴定位：感知形态[语言型 + 视觉型] × 传播动力[荒诞共谋]
交互隐喻：完成今日摸鱼指标 = 用你最讨厌的考核形式，反过来考核你够不够松弛
视觉意象：一个和钉钉/飞书绩效面板一模一样的仪表盘，但指标是「今日发呆 0.5/2 小时 🔴 未达标」「本周摸鱼完成率 37%」，红字飘着「请尽快约谈自己」
情绪时刻：打开看到自己「摸鱼 KPI 严重不达标」，先一愣——我连摆烂都没达标？然后笑出来。

【方向 2 · 浪费时间处方】
思维方法：概念错位（把「医生处方 / 医嘱」这种不容置疑的医疗权威，搬到日常摸鱼场景）
双轴定位：感知形态[语言型] × 传播动力[反差击中]
交互隐喻：领取今日处方 = 让一个「医生」一本正经地开出胡闹医嘱，于是偷懒变成了「遵医嘱」
视觉意象：一张仿三甲医院的处方笺，盖着红章，写着「诊断：用力过猛。处方：饭后发呆 15 分钟，每日 3 次；睡前躺平刷无意义短视频 1 剂；忌：打开待办清单。」
情绪时刻：拿到一张盖红章的正经处方，写着「忌打开待办清单」，那种「居然有人给我开假条」的好笑。

【方向 3 · 反向番茄钟】
思维方法：形态映射（把情绪内核映射到时间型——番茄钟原本逼你专注，整个反过来）
双轴定位：感知形态[时间型 + 身体型] × 传播动力[被代言感]
交互隐喻：启动倒计时 = 这 25 分钟你被禁止做任何有用的事，专注，是专注地浪费
视觉意象：一个和经典番茄钟一样的圆形倒计时，但中间写「禁止高效 24:59」，你一打开待办类 App 它就「啪」地暂停并扣分
情绪时刻：倒计时跳动，你居然有点紧张——这 25 分钟我必须浪费够，不许干正事。

【方向 4 · 今天我浪费了什么】
思维方法：触发反推（先锁定传播动作「我也要晒一个我的版本」=参与冲动，再反推出打卡晒图的形态）
双轴定位：感知形态[视觉型 + 语言型] × 传播动力[参与冲动]
交互隐喻：记录今日浪费 = 把「无意义」郑重其事地存档，无意义本身成了内容
视觉意象：一个像 Keep / 读书打卡那样的成就卡片，但内容是「今日浪费成就：盯着天花板 22 分钟 🏆」，一键生成可分享长图
情绪时刻：把「发呆 22 分钟」做成一张闪闪发光的成就卡，那种「我居然为废物时间颁了个奖」的爽。

【方向 5 · 浪费收据】
思维方法：自由漂移（随机词碰撞：抓「收据 / 小票」这个和情绪内核完全无关的物件，硬连——浪费的时间能不能像消费一样打一张小票？不参照任何案例）
双轴定位：感知形态[视觉型] × 传播动力[荒诞共谋]
交互隐喻：打印浪费小票 = 给「什么都没产出」开一张正式凭证，让虚度有了实体
视觉意象：一张超市热敏小票的样式，逐行打印「09:14 刷手机 ¥0.00」「14:30 发呆 ¥0.00」，底部「合计：今日虚度 2 小时 31 分，谢谢惠顾」，可以一直攒成一个月的长卷
情绪时刻：撕下一张写满「¥0.00」的废物小票，荒诞又解压，想拍下来发群里。

【方向 6 · 越努力越报警】
思维方法：自由漂移（反转 + 极端化：把「提醒你去浪费」整个反过来——不是劝，是监测；你越卷它越受不了，推到最极端的对抗）
双轴定位：感知形态[身体型 + 声音型] × 传播动力[压不住的笑]
交互隐喻：检测到你过度高效 = 一个比你还轴的系统强行喊停你，努力变成了违章
视觉意象：一个像烟雾报警器/血氧仪的小挂件（或手机后台），监测到你连续高效工作 90 分钟，突然「滴——滴——」红灯狂闪，屏幕黑掉弹「检测到过劳，已为您强制休息」
情绪时刻：正干得起劲，机器突然像抓现行一样报警「你太卷了」，气笑了，但真的停了下来。

【方向 7 · 摸鱼人格测试】
思维方法：案例锚定（参考案例库 SBTI「测出自己是吗喽却骄傲转发」的自嘲手法，做同类）
双轴定位：感知形态[语言型] × 传播动力[荒诞共谋 + 身份归属]
交互隐喻：完成测试 = 给自己贴一个浪费时间的荒诞工种标签，并骄傲认领
视觉意象：仿正经职业测评的界面，结果却是「你的摸鱼人格：资深天花板凝视师 / 带薪发呆架构师 / 厕所续命工程师」，配一张煞有介事的「职业认证书」
情绪时刻：测出来自己是「资深天花板凝视师」，配认证书，那种「这破称号我居然想印名片上」的骄傲。

【方向 8 · 每日功德 · 浪费版】
思维方法：概念错位（把「积功德 / 修行打卡」这种宗教-修行语境，搬到「修一个虚度成正果」的反语境）
双轴定位：感知形态[视觉型 + 语言型] × 传播动力[荒诞共谋]
交互隐喻：每日积「废功德」= 把无所事事修成一种正经的功课，越虚度越得道
视觉意象：一个木鱼/电子佛龛界面，敲一下「功德 +1」，但功德事项是「今日已虚度，距离『躺平成佛』还差 880 功德」，年底给你颁「年度废物·上上签」
情绪时刻：敲一下木鱼，弹出「虚度功德 +1，善哉」，无厘头到想拉朋友一起敲。

> 质量自检：8 个方向含「概念错位」3 个（1、2、8）、「自由漂移」2 个（5、6），满足下限；全程守住轻/荒诞的语气，没有把任何一个升华成「关爱自己」「对抗异化」的沉重表达；每个方向双轴齐全。

哪 1–2 个方向最让你有感觉？

## 概念卡 A · 越努力越报警

**① 产品名 + 一句话定义**
候选名：**过劳报警器 / OverWork Alarm** ｜ **卷烟报警器**（谐音「检测你冒没冒『卷』烟」）｜ **太努力了 警告**
一句话定义：一个你**越努力它越受不了、会当场抓你现行**的报警器。

**② 核心情绪时刻**
正干得起劲，机器突然像抓现行一样「滴——滴——」红灯狂闪，屏幕黑掉甩出一句「检测到过劳，已为您强制休息」——你被气笑了，但手真的停了下来。

**③ 交互隐喻**
检测到你连续高效 = 努力变成了一种「违章」，一个比你还轴的系统替你喊停。
（动词等式：**持续高效工作 = 触发警报，被强制摸鱼**）

**④ 视觉方向**
- 色彩基调：安全警示黄 + 警报红 + 工业黑（像消防报警器/施工警示牌）
- 核心视觉动作：进度条悄悄涨满后，整屏「红黑频闪」+ 横幅式警告滑入，像系统蓝屏那样毫无商量
- 情绪氛围：一本正经的工业级严肃，配上荒唐的报警理由，反差越硬越好笑

**⑤ Vibe Coding 提示词**
```
过劳报警器（OverWork Alarm）交互原型

视觉风格：industrial warning, hazard yellow & alarm red, blinking siren, system-alert banner, monospaced
核心动效：后台计时连续高效时长，进度满后整屏红黑频闪 + 顶部滑入警告横幅 + 短促「滴滴」蜂鸣，屏幕变灰锁定 N 分钟倒计时
技术方向：Web 前端（React + CSS keyframes 频闪 + Web Audio API 蜂鸣）即可；可选包成桌面/手机后台监测
情绪氛围：deadpan-serious, absurd, laugh-out-loud
核心交互逻辑：用户开始工作即计时；连续高效达阈值（如 90 分钟无中断）触发全屏报警，强制锁屏进入「摸鱼倒计时」，期间打开任何工作类页面会延长锁定；倒计时结束才解禁。

请构建一个可在浏览器中运行的交互原型，重点实现「连续高效 → 全屏红黑频闪报警 → 强制摸鱼锁屏倒计时」，整体风格参考工业警示牌，让用户感受到「正干得起劲被机器抓现行、气笑了但真停下来」。
```

**⑥ 传播钩子**（2–3 候选）
- 「全网第一个**你越努力它越生气**的 App」
- 「检测到过劳，已为您强制休息。」（直接把报警语当置顶）
- 「别人的闹钟叫你起床，我的闹钟叫我别干了。」

## 概念卡 B · 浪费时间处方

**① 产品名 + 一句话定义**
候选名：**摸鱼处方笺 / Rx: Chill** ｜ **虚度医嘱** ｜ **闲诊所**
一句话定义：一个每天**一本正经给你开「浪费时间医嘱」**的电子诊所。

**② 核心情绪时刻**
拿到一张盖着红章的正经处方，上面写着「忌：打开待办清单」「饭后发呆 15 分钟，每日 3 次」——那种「居然真有人给我开摸鱼假条、还盖了章」的好笑。

**③ 交互隐喻**
领取今日处方 = 让一个不容置疑的「医生」下医嘱，于是偷懒摇身变成「遵医嘱」。
（动词等式：**领取处方并执行 = 把放纵升级成不可违抗的医疗指令**）

**④ 视觉方向**
- 色彩基调：处方笺米白 + 印章红 + 钢笔墨蓝
- 核心视觉动作：每日生成一张处方笺，红章「啪」地盖下（带轻微旋转回弹），手写体逐字落笔
- 情绪氛围：三甲医院的不容置疑感，配上胡闹的医嘱，越像真的越好笑

**⑤ Vibe Coding 提示词**
```
摸鱼处方笺（Rx: Chill）交互原型

视觉风格：medical prescription, off-white paper, red official stamp, handwritten script font, serious clinical layout
核心动效：进入即抽取今日处方，红章盖下的「啪 + 回弹」动画 + 医嘱逐字手写浮现；可一键「续方/换一张」
技术方向：Web 前端（React + CSS transform 盖章动效 + 打字机文字），处方内容用模板库随机组合
情绪氛围：deadpan, absurd, oddly-official
核心交互逻辑：每日生成一张随机「胡闹医嘱」处方（诊断 + 用法用量 + 禁忌），用户可「遵医嘱」打卡或「换一张」，长按可生成可分享的处方长图。

请构建一个可在浏览器中运行的交互原型，重点实现「抽取处方 → 红章盖下 → 医嘱逐字浮现 → 一键分享」，整体风格参考真实医院处方笺，让用户感受到「居然有人正经给我开摸鱼假条」的好笑。
```

**⑥ 传播钩子**（2–3 候选）
- 「医生，我想请个『不上进』的假。——准。」
- 「处方：饭后发呆 15 分钟，每日 3 次。忌：打开待办清单。」
- 「这是我今年唯一遵医嘱的一次。」
